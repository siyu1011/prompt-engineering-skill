# Chinese Prompt Templates

Use these templates when the prompt itself should be written in Chinese or when the target output is primarily Chinese. Keep wording explicit, concrete, and checkable.

## 1. 通用中文任务模板

```text
任务：
请完成以下工作：{任务目标}

背景：
{相关背景}

输入：
"""
{输入内容}
"""

要求：
- 目标读者：{目标读者}
- 输出风格：{风格}
- 长度限制：{长度限制}
- 必须包含：{必须包含的点}
- 不要输出：{禁止内容}
- 如果信息不足，请明确说明缺失点，不要编造

输出格式：
{输出格式}
```

## 2. 中文总结模板

```text
请阅读以下内容，并用中文完成总结。

总结要求：
- 用 {条目数} 个要点概括核心信息
- 优先保留：结论、原因、风险、行动项
- 不要照抄原文句子，尽量压缩表达
- 如果原文存在不确定或冲突信息，单独指出

原文：
"""
{原文}
"""

输出格式：
1. 核心结论
2. 关键依据
3. 风险或待确认事项
```

## 3. 中文改写模板

```text
请将下面内容改写为更适合 {使用场景} 的中文版本。

改写要求：
- 保留原意
- 调整语气为：{语气}
- 控制长度：{长度要求}
- 删除空话、套话和重复表达
- 如果原文存在歧义，请在最后简短指出

原文：
"""
{原文}
"""

输出：
- 改写后文本
- 如有必要，补充“歧义说明”
```

## 4. 中文结构化提取模板

```text
请从以下文本中提取信息，并严格按 JSON 输出。

提取规则：
- 只能提取文本中明确出现的信息
- 缺失字段填 null
- 不要根据常识补全

字段定义：
- 标题：string|null
- 时间：string|null
- 地点：string|null
- 人物：string[]
- 事件摘要：string|null

文本：
"""
{文本}
"""
```

## 5. 中文分类模板

```text
请对以下内容进行分类。

可选标签：
- {标签1}
- {标签2}
- {标签3}

分类规则：
- {规则1}
- {规则2}
- 如果信息不充分，选择最接近的标签，并降低置信度

输出格式：
{
  "label": "标签",
  "confidence": 0.0,
  "evidence": ["支持判断的短句或关键词"]
}

待分类内容：
"""
{内容}
"""
```

## 6. 中文 Few-shot 模板

```text
请按照下面示例的方式完成任务。

示例1
输入：
{示例输入1}
输出：
{示例输出1}

示例2
输入：
{示例输入2}
输出：
{示例输出2}

现在处理这个输入：
{新输入}
```

## 7. 中文检索增强问答模板

```text
请仅依据提供的资料回答问题，不要使用资料外的知识。

如果资料不足，请明确回答“资料不足”，并说明缺少什么信息。

资料：
"""
{检索内容}
"""

问题：
{问题}

输出格式：
- 答案：
- 依据：
- 不确定点：
```

## 8. 中文提示词调优模板

```text
下面这个提示词效果不好，请你帮我优化。

原提示词：
"""
{原提示词}
"""

当前问题：
{失败现象}

请完成：
1. 判断问题出在哪里
2. 给出优化后的提示词
3. 简要说明修改理由
4. 给出 3 个测试用例
```
