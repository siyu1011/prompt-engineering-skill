# Dify Integration

This directory contains Dify-ready prompt variants.

Recommended usage:

1. Use `system-prompt.md` as the agent's system prompt.
2. Use `opening-message.md` as the opening message if needed.
3. Optionally adapt the content to Dify variables, workflows, or knowledge retrieval.

Notes:

- Dify supports multiple agent and workflow styles.
- The files here are written as a strong general-purpose baseline rather than a workflow-export format.
