# Cursor Integration

This directory contains Cursor-oriented rule files.

Recommended usage:

1. Copy `cursor-prompt-engineering.mdc` into your Cursor rules setup if you use rule files.
2. Or paste the contents into project-level AI rules or instructions.
3. Use the toolkit files under `../../prompts/` and `../../guides/` as reference material.

Notes:

- Cursor setups vary by version and workflow.
- The file here is written as a portable rule document rather than assuming one exact Cursor feature.
