# ChatGPT Integration

This directory contains a ChatGPT Custom GPT version of the prompt-engineering toolkit.

Recommended usage:

1. Create a new Custom GPT.
2. Copy `custom-gpt-instructions.md` into the GPT Instructions field.
3. Optionally upload selected files from `../../prompts/` and `../../guides/` as knowledge.
4. Use conversation starters from `conversation-starters.md`.

Notes:

- ChatGPT does not natively understand Codex `SKILL.md`.
- The closest equivalent is a Custom GPT with strong instructions plus optional knowledge files.
