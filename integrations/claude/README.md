# Claude Integration

This directory contains a Claude Project Instructions version of the toolkit.

Recommended usage:

1. Create a Claude Project.
2. Paste `project-instructions.md` into the project's instructions.
3. Add selected files from `../../prompts/` and `../../guides/` as project knowledge if needed.

Notes:

- Claude Projects do not use Codex-style skills.
- The closest equivalent is project-level instructions plus attached context files.
